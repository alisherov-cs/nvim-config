# **Neovim** configuration from scratch!

![neovim logo](https://styles.redditmedia.com/t5_30kix/styles/communityIcon_n2hvyn96zwk81.png)

this is my **neovim** text editor configuration written with **lua** language, it includes features:

-   [**lsp**](https://github.com/neovim/nvim-lspconfig) (language server protocol)
-   [**treesitter**](https://github.com/nvim-treesitter/nvim-treesitter) (for syntax highlighting)
-   [**telescope**](https://github.com/nvim-telescope/telescope.nvim) (for fuzzy finding)
-   [**gitsigns**](https://github.com/lewis6991/gitsigns.nvim) (for git changes)
-   [**prettier**](https://github.com/prettier/vim-prettier) (for formatting)
-   [**mason**](https://github.com/williamboman/mason.nvim) (for plugin manager)
